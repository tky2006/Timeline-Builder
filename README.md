## 🧠 About the Game

**Timeline Builder** challenges players to correctly arrange historical events in chronological order (oldest → newest). Each round presents a shuffled set of Malaysian historical events, and players must drag them into the correct sequence before time runs out.

---

## ✨ Features

- ⏳ **30-second timed rounds**
- ❤️ **3 lives system**
- 🏆 **Score tracking (10 points per correct round)**
- 📊 **10 progressive rounds**
- 🎯 **Drag & drop timeline mechanics**
- 🔊 **Sound effects + background music**
- 📱 **Responsive design (mobile friendly)**
- 🎨 Clean, themed UI inspired by Malaysian heritage aesthetics

## 📚 Educational Content

Covers key Malaysian history topics including:

- Federation of Malaya (1948)
- Independence of Malaya (1957)
- Formation of Malaysia (1963)
- Singapore separation (1965)
- ASEAN formation (1967)
- May 13 Incident (1969)
- New Economic Policy (NEP)
- Rukun Negara

## 🕹️ How to Play

1. Click **START**
2. Look at the shuffled historical events
3. Drag each event into the correct timeline order
4. Arrange from **oldest → newest**
5. Submit automatically when all slots are filled
6. Earn points for correct answers
7. Avoid losing all 3 lives!

---

## 🧩 Game Rules

- ✔ Correct timeline = **+10 points**
- ❌ Wrong timeline = **-1 life**
- ⏰ Time out = **-1 life**
- 🏁 Game ends when lives reach 0 or all rounds are completed
